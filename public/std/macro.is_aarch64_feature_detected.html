Expand description

This macro tests, at runtime, whether an aarch64 feature is enabled on aarch64 platforms. Currently most features are only supported on linux-based platforms.

This macro takes one argument which is a string literal of the feature being tested for. The feature names are mostly taken from their FEAT_* definitiions in the ARM Architecture Reference Manual.

Supported arguments

  • "asimd" or “neon” - FEAT_AdvSIMD
  • "pmull" - FEAT_PMULL
  • "fp" - FEAT_FP
  • "fp16" - FEAT_FP16
  • "sve" - FEAT_SVE
  • "crc" - FEAT_CRC
  • "lse" - FEAT_LSE
  • "lse2" - FEAT_LSE2
  • "rdm" - FEAT_RDM
  • "rcpc" - FEAT_LRCPC
  • "rcpc2" - FEAT_LRCPC2

  • "pauth" - FEAT_PAuth
